# VTT Slicer

VTT Slicer is a browser-based tool for slicing tabletop RPG map images into printable US Letter PDF pages.

## Goals
- Upload a map image
- Set final printed size in inches
- Slice into 8x10 or 8x10.5 page tiles
- Add optional grid overlay
- Export a multi-page PDF for home printing
